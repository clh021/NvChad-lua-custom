# NvChad-lua-custom
NvChad custom config

```vimCommand
# 查看/导出 所有快捷键配置
:nmap for normal mode mappings
:vmap for visual mode mappings
:imap for insert mode mappings
# 导出
:redir! > nvchad.keymap.txt
:silent verbose map
:redir END
:pwd
```
